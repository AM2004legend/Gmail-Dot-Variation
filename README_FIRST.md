## What This App Does

This app generates **Gmail address variations** using Gmail's dot naming feature:

- `user@gmail.com` 
- `u.ser@gmail.com` ‚Üê (generates these)
- `us.er@gmail.com`
- `use.r@gmail.com`
- etc.

Then **saves and manages** your collection of variations.
